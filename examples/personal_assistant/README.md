# Example: Personal Assistant with Profiling / 带画像的个人助理

This example demonstrates **long-term memory + user profiling**.

本示例展示 **长期记忆 + 用户画像**。

## Scenario / 场景

The assistant remembers user preferences across sessions:
- Preferred communication style (formal / casual)
- Frequently discussed topics
- Work habits and scheduling preferences

The `DreamEngine` periodically consolidates these traits in the background.

## Run / 运行

```bash
pip install -r requirements.txt
python main.py
```
